# it-talk
关于it的一切讨论,提问/问答

## 使用

1. 如果有IT相关的疑问,如设计、实现、求职、offer选择等,请通过**issue**的方式提问.


## 声明

1. 项目本身不涉及技术实现,只作为IT从业者**问题讨论**的空间.
2. 只讨论和**IT技术**有关的问题, 不要水帖
3. 1) 该项目的维护人员也在国内一线大厂上班,所以可能不会即时回复,请见谅 :)
   2) 能力有限, 不可能全部能回答所有的疑问; 互相探讨 :)
4. 希望更多的大牛参与问题的讨论
5. 技术讨论,请注意措辞和语气
